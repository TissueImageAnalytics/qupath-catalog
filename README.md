# qupath-catalog

QuPath extension catalog from the TIA Centre, University of Warwick. Currently lists the [TIAToolbox QuPath extension](https://github.com/TissueImageAnalytics/tiatoolbox-qupath-extension), which runs [TIAToolbox](https://github.com/TissueImageAnalytics/tiatoolbox) models from inside [QuPath](https://qupath.github.io/).

## Adding it to QuPath

In QuPath, open **Extensions → Manage extensions → Add catalog** and paste:

```
https://github.com/TissueImageAnalytics/qupath-catalog
```

The extension will appear in the list. Install it and restart QuPath.
